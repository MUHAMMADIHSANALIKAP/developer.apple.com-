    Description

    Turn your phone into an FTP server

    The FTP server allows the FTP clients to connect to the phone.
    Enter the port number, username and password - you're done
    The following are displayed:
    -number of open communication ports
    -broadcast preview
    -files are easy accessible from Files app in FTP-Server directory
