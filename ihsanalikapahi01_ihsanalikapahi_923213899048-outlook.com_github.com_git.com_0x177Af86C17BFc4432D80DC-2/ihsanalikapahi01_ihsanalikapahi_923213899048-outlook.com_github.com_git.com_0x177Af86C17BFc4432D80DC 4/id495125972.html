    Critics remorse , 09/25/2017

    Unsatified

    Hi. I like this app for all of its conveniences and I enjoy the many notes that I was able to take on the app. However, now I’m displeased with the fact that I have many work documents that I can not access due to loading the latest upgrade and that makes me mad. You should at least let people know that due to the update we will not be updating this app, so they can retrieve their documents. The mutual bond that a developer has with a client or customer is based off of trust and I trusted that using this app would help and not hinder me from reaching my goals and I can’t even access the documents I had saved. EXTREMELY DISAPPOINTED😑!

    SpongeGarFweeFwee , 06/23/2016

    Good, but a few problems

    I really love this app and use it for notes all the time, but there are a few big problems. Problem A, Emojis/FancyKey. The app won't let me use and alternate keyboards or I have to go out of the note & back in. It's really annoying! Problem B (Which is the more tedious of the issues), self-deleting notes. Occasionally when I try to enter a note to edit it, it completely erases itself. Instead of showing the original note, the note is blank with the note titled as 'Untitled'. I've lost at least a dozen very important notes to this, it's terrible!!

    Please fix these things!!

    Willow Cove , 02/15/2014

    Very Good App

    As a couple of other reviewers have already mentioned, I too downloaded this app after having lost important entries in I-pad Notes. I really like this app. It performs exactly as stated, and is easy to use. My only suggestion for improvement is to add a rich-text type feature so that font size can be changed, and both bold and italicized text can be used.
